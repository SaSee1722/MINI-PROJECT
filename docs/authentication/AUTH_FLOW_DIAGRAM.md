## ğŸ”‘ Key Points

1. **Two Ways to Auth**:
   - Use `useAuth()` hook in React components
   - Use utility functions from `authFunctions.js` anywhere

2. **Automatic Profile Creation**:
   - Database trigger creates user profile on signup
   - Profile includes: id, email, name, role

3. **Role-Based Access**:
   - Admin: Full access to all features
   - Staff: Limited to attendance marking

4. **Session Persistence**:
   - Sessions stored in browser
   - Auto-restored on page reload
   - Managed by Supabase

5. **Security**:
   - Passwords hashed by Supabase
   - RLS policies protect data
   - Email verification (optional)
   - Secure session tokens
